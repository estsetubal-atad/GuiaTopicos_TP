### Exposição

### Exploração por LLM

> Na aula anterior já vimos alguns tipos primitivos, e.g., `int` e `float` e derivados, e.g., *arrays* de inteiros.

**Prompt inicial:**

```console
Nos problemas seguintes, dá-me um exemplo de resolução completo na linguagem C utilizando técnicas básicas de programação. Se necessitar de explicação de código, irei pedi-te. Não comentes o código.
```

**Prompt 1:**

```console
Gera-me uma tabela resumida dos tipos de dados primitivos da linguagem C.
```

**Prompt 2:**

```console
Mostra-me duas formas de inicializar uma string em C com o texto "Creativo".
```

**Prompt 3:**

```console
Como calcular o tamanho de uma destas strings em tempo de execução?
```

> As *strings* em C não são objetos, logo não temos métodos, e.g., `s.length()`.

**Prompt 4:**

```console
Mostra-me um programa em C que solicita ao utilizador o seu nome completo e utiliza uma função para substituir ocorrências A por 4 e O por 0, mostrando o resultado.
```

> A solução apresentada poderá carecer de questões de explicação.  
> Podem ser discutidas implementações alternativas.

**Prompt 5:**

```console
Mostra-me um programa em C que defina um tipo composto Livro, contendo autor, título e ano de publicação.
```

> Como passar a definição de Livro para um módulo?

**Prompt 6:**

```console
Modifica o programa anterior por forma a utilizar duas funções para inicializar e imprimir o conteúdo de uma variável do tipo Livro. Não utilizes ponteiros.
```
