## 1 - Introdução à Linguagem C

### Exposição

Slides `3 ~ 6`.

### Exploração por LLM

> Em IPOO já utilizaram o método `System.out.printf` e os especificadores de formato.

**Prompt inicial:**

```console
Nos problemas seguintes, dá-me um exemplo de resolução completo na linguagem C utilizando técnicas básicas de programação. Se necessitar de explicação de código, irei pedi-te. Não comentes o código.
```

**Prompt 1:**

```console
Mostra-me um programa em C que calcule a média de 3 números inteiros e imprima o resultado com uma casa decimal. Os valores inteiros não são solicitados ao utilizador, são literais.
```

**Prompt 2:**

```console
Mostra-me um programa em C que solicite ao utilizador o valor do lado e altura de um rectângulo (inteiros). A saída do programa deverá indicar se se trata de um rectângulo ou quadrado, assim como a sua área.
```

**Prompt 3:**

```console
Modifica o programa anterior por forma a utilizar uma função para calcular a área.
```

**Prompt 4:**

```console
Mostra-me um programa em C que imprime na consola os números pares entre 1 e N (valor solicitado ao utilizador) e a sua contagem. Caso N seja negativo, informa o utilizador e nada faz.
```

> Existe alguma forma de eliminar o `else`? 

**Prompt 5:**

```console
Mostra-me um programa em C que instancie um array de 5 inteiros à tua escolha e que calcule a sua soma.
```

> Não temos `.length`, logo esta informação tem sempre de existir em "algum lado".

**Prompt 6:**

```console
Modifica o programa anterior por forma a utilizar uma função para calcular a soma dos elementos.
```
