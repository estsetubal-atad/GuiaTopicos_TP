## 8 - Introdução aos Tipos Abstratos de Dados

### Exposição / Discussão / Exploração por LLM

Slides `1 ~ 35`

> Os slides contêm questões para discussão em aula de pesquisas ou exploração por LLM de conceitos. Transcrevem-se de seguida as *prompts* existentes nos slides por facilidade de uso, que deverão ser utilizadas para responder às questões nos slides:

**Pesquisa (google):**

```console
define: abstração
```

**Prompt 1:**

```console
Assume que sou um estudante de engenharia informática no meu primeiro ano de curso. 
Define "tipo abstrato de dados" de uma forma simples.
```

**Prompt 2:**

```console
Qual a diferença entre "abstração de processos" e "abstração de tipos"?
```

**Prompt 3:**

```console
No contexto dos tipos abstratos de dados, o que significam "especificação", "implementação" e "utilização"?
```

**Prompt 4:**

```console
Conheço a linguagem Java. 
Como é que um tipo é abstraído nesta linguagem e como é feito o controlo de erros?
```

**Prompt 5:**

```console
Como é que isto será possível na linguagem C?
```
